About LearnTrack

Built to visualize every student's journey

LearnTrack is a digital platform designed to monitor and visualize academic progress across every semester of a student's school life.

Our Mission

We believe every student deserves to see their growth clearly. LearnTrack bridges the gap between raw grades and meaningful insights, turning numbers into a story of progress.

Our platform enables teachers to upload grades seamlessly while empowering students to understand their trajectory, set goals, and reflect on what matters most.

Two Roles, One Platform
Built for everyone in the classroom

For Students

View grade progress, understand your learning status, write personal reflections, and stay motivated.

For Teachers

Upload class grades via Excel, monitor every student's performance, add notes to reflections.

Free Deployment

Deploy to GitHub Pages

Rename your file to index.html and follow these steps — your LearnTrack site will be live with a public URL in under 2 minutes, for free.

1

Create a GitHub repository

Go to github.com/new, name it learntrack, set visibility to Public, then click Create repository.

2

Rename file to index.html

GitHub Pages serves index.html as the homepage. Rename learntrack.html → index.html before uploading.

3

Upload the file

In your repository, click Add file → Upload files, drag your index.html in, write a commit message (e.g. "initial upload"), then click Commit changes.

4

Enable GitHub Pages

Go to Settings → Pages. Under Source, choose Deploy from a branch, select main branch + / (root), then click Save.

5

Visit your live site!

After ~1 minute your site is live at:

https://your-username.github.io/learntrack/

Find the exact link in Settings → Pages anytime.
